📄 Visa Cover Letter Generator (Python & Streamlit)
A professional tool designed to automate the generation of high-quality visa cover letters for Schengen and international travel. This application specifically generates a detailed itinerary and professional profile based on the travel history and business background of Rana Muhammad Asif.  

🚀 Features
Automated Document Creation: Generates a complete .docx file with professional formatting.  

Detailed Itinerary: Includes a comprehensive 10-day plan for Belgium, covering iconic spots like Grand Place, Atomium, and day trips to Bruges and Ghent.  

Professional Profiling: Integrates business details for RANA BUILDERS (NTN: 2815574-2) and memberships with the Islamabad Chamber of Commerce.  

Extensive Travel History: Highlights a record of visits to 15+ countries including the UK, USA, Saudi Arabia, and Turkey.  

Modern UI: Built with Streamlit for a clean and easy-to-use user interface.

🛠️ Technology Stack
Python 3.14+

Streamlit: For the web interface.

python-docx: For creating and formatting Word documents.

BytesIO: For handling in-memory file generation and downloads.

📋 Prerequisites
Make sure you have Python installed. You can install the required libraries using pip:

Bash
pip install streamlit python-docx
🏃 How to Run
Clone this repository to your local machine.

Navigate to the folder:

Bash
cd "Ai Cover Letter"
Run the application:

Bash
streamlit run visa_app.py
📂 Project Structure
visa_app.py: The main application script containing the UI and document logic.
